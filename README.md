# Healthy Latina Lifestyle Mobile App

#### Features for Prototype:
* Create an account
* Change password
* Update profile information
* Delete account
* View home page
* List all meditations
* View meditation page
* Create a meditation page
* Filter for meditations
  * Difficulty
  * Time
  * Topic
  * Language
* Built-in Timer
